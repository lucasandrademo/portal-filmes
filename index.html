Sobre

Esse site de notícias sobre filmes foi criado pelo Aluno Lucas Machado de Oliveira Andrade, para a disciplina de Desenvolvimento de Interfaces WEB (DIW), sendo que essa disciplina é ministrada pelos professores Rommel Vieira Carneiro e João Caram.
Para o desenolvimento desse trabalho foi utilizado apenas tecnologias front-end, sendo elas, o uso de imagens, linguagens CSS e HTML, media querys, bootstrap e font awesome.
O bootstrap foi utilizado como framework principal para efetuar o trabalho de forma responsiva, enquanto o font awesome foi utilizado como fonte de ícones principal do trabalho, pois é gratuito, de fácil implementação e utilização.
O sistema criado é responsivo e capaz de atualizar os dados via API themoviedb.
